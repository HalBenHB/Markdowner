### Instructions

1.  **Data Collection:**
    *   Go to [investing.com](https://www.investing.com) and search for **Colgate-Palmolive (Ticker symbol: CL)**.

    *   *Hint:* Don’t forget to set the time frame to “Monthly” and adjust the time range accordingly. Download the CSV file and import it to Excel.
2.  **Calculation:**

### Tasks

1.  **Graphing:** Create a line graph in Excel where you put “monthly date” on the x-axis and “return” on the y-axis.
2.  **Analysis:** Compute the average return and standard deviation for each year (from 2020 to 2024).
    *   *Hint:* Prepare a table with three columns:

        *   **Column 2:** Average return (in percentage)
